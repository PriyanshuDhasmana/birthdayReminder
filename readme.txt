Birthday Reminder Application (Python GUI)

A desktop-based birthday reminder application built using **Python & Tkinter**.

Features
- Calendar-based birthday selection
- Reminder alert **one day before** the birthday
- Customizable reminder time (default: 11:58 PM)
- Custom alarm sound selection via UI
- Persistent storage using JSON
- Background reminder checker
- Clean & modern dark UI

Tech Stack
- Python 3
- Tkinter
- tkcalendar
- playsound

Installation
```bash
pip install -r requirements.txt
